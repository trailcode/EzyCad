# Icon sources (FreeCAD Draft, LGPL-2.0)

SVGs can be downloaded from the FreeCAD repo and converted to PNG for the toolbar.

## Shape duplicate modes

| EzyCad mode              | FreeCAD icon (SVG)   | Raw URL |
|--------------------------|----------------------|---------|
| Shape_rect_duplicate     | Draft_Array.svg      | https://raw.githubusercontent.com/FreeCAD/FreeCAD/main/src/Mod/Draft/Resources/icons/Draft_Array.svg |
| Shape_path_duplicate     | Draft_PathArray.svg  | https://raw.githubusercontent.com/FreeCAD/FreeCAD/main/src/Mod/Draft/Resources/icons/Draft_PathArray.svg |

Full Draft icons: https://github.com/FreeCAD/FreeCAD/tree/main/src/Mod/Draft/Resources/icons

Convert SVG → PNG at **48×48** (e.g. Inkscape, ImageMagick) and save as `Draft_Array.png`, `Draft_PathArray.png` in this folder. Toolbar buttons display at 48×48.
